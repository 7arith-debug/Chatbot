HarithChatbot is a desktop-based chatbot application developed using Java Swing. The chatbot allows users to interact through a graphical interface and provides responses using offline keyword-based logic. The project was developed as part of an In-House Internship Project to demonstrate concepts of Java GUI programming, event handling, multithreading, and basic chatbot design.

This application focuses on simplicity, usability, and modular design, making it suitable for beginners and academic purposes.
